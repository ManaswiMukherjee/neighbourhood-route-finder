# Neighbourhood-route-finder

<h2>Overview:</h2>
This project aims to implement the shortest route for local public transport vehicles to reduce costs by finding the shortest route between two selected points.

<h2>Features:</h2>
Terminal application with the ability to select download and local modes and a gui location selector.

<h2>Tools used:</h2>
Python 3.13.9
OSMnx - 2.0.6
matplotlib - 3.10.7

VS-code

<h1>Steps to install and run the application:</h1>

1. Clone the repository

`git clone https://github.com/ManaswiMukherjee/neighbourhood-route-finder.git `

2. cd into the repo
`cd neighbourhood-route-finder`

3. create and activate virtual environment\

**for macOS/Linux:**

`python -m venv .venv`\
`. .venv/bin/activate`

**Windows(PowerShell):**

`python -m venv .venv`\
`.\.venv\Scripts\Activate.ps1`

4. Install runtime dependencies

`pip install -r requirements.txt`
